Welcome to Minesweeper

Minesweeper is a logic game for single player. Main part of the game is board with defined number of fields that contains hidden "mines". The goal of the game is to uncover all safe (blank) fields on board without detonating any of hidden "mines" in the quickest time possible. Fields contain clues about the number of neighboring mines in each field.

About Minesweeper

Minesweeper is a logic game for single player. Main part of the game is board with defined number of fields that contains hidden 'mines'. The goal of the game is to uncover all safe (blank) fields on board without detonating any of hidden 'mines' in the quickest time possible. Fields contain clues about the number of neighboring mines in each field.

Game History

The game originates from the 1960s. However, the game we know today was created by Robert Donner and Curt Johnson both Microsoft employees in 1989. First it was officially released as part of the Windows Entertainment Pack in 1990. Finally, in 1992. the game became part of Microsoft operating system Windows 3.1.

Game Rules

  • Game starts when player clicks on first field, and that also starts the timer.
  • First field is always "safe" meaning it can't be a mine.
  • Player can flag a potential mine so he can navigate easier.
  • Player loses game if opens field that contains mine.
  • Player wins if opens all of the "safe" fields.
  • Players are ranked based on time spent to finish the game.
